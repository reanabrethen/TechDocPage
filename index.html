Attributes

HTML attributes provide additional information about elements

attributes

There are several different attributes for each element in HTML. They usually come in name & value pairs (ex.: name="value")

  • The "a" tag defines a hyperlink (a clickable link within text). The "href" attribute is used to specify the URL the link will go to
  • The "img" tag is used to embed an image within an HTML page. The "src" attribute is used to specify the path & will display the image on the web browser
  • The "html" tag will use the "lang" attribute to define the language used on a Web page. This can change via the language used for the page itself (ex.: lang="en" for English language used on webpage)
  • The "title" attribute is used to provide more information about an element (ex. title="Lorem, ipsum.")
